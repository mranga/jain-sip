Summary Changes from version 1.2 of this specification

  1. The following classes have been introduced in version 2.0 of this package :
    • JoinHeader : See RFC 3911
    • MinSEHeader : See RFC 4028
    • PathHeader : See RFC 3327
    • ReferredByHeader : See RFC 3515
    • ReplacesHeader: See RFC 3891
    • SessionExpiresHeader: See RFC 4028
  2. Support for these headers and all the headers defined in javax.sip.header.ims have been added to HeaderFactory
